## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites
IMPORTANT: you need to change your system locale to ```en-us``` if you use ```Export results to Excel``` feature.

What things you need to install to run the application.

```
Microsoft Visual Studio 15++
MetroSet_UI 1.0.4
Microsoft Office Interop Excel 15.0.4795.1000
.NET Framework 3.5
```

Also, you need to run the ```.sql``` file located at ```SQL``` folder before running the application.

## Deployment
Open Microsoft Visual Studio 15, open the project and click the GREEN forward arrow :arrow_forward: to build and run the application.
